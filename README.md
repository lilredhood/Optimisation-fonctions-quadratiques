# Optimisation-fonctions-quadratiques
Projet d'optimisation de fonctions quadratiques avec différentes méthodes de descente de gradient (https://fr.wikipedia.org/wiki/Algorithme_du_gradient) :
- gradient à pas constant 
- gradient à direction conjugué
- gradient à pas optimale
- gradient à pas optimale, version avec des matrices creuses (format CSR https://de.wikipedia.org/wiki/Compressed_Row_Storage)
